# void-patrol

Void Patrol is a vertical-scrolling top-down space shooter where the player pilots a fighter through 6 procedurally-ramped enemy waves, then faces a 3-phase boss with a weak point. Power-ups drop from tougher enemies (double/triple shot, laser, shield restore, speed, bomb), score and high score persist across runs, and each cleared loop bumps difficulty by 1.

## Features
- Player ship & combat core
- Bullets system
- Enemy types & AI
- Procedural wave manager
- Boss fight
- Power-ups
- Game flow, scoring, high score
- Visual polish & audio
